# Week 1 - Tokenization and Bigram Baseline

## Materi yang Dipelajari

- Character Vocabulary
- STOI (String to Integer)
- ITOS (Integer to String)
- Encode & Decode
- Token Frequency
- Bigram Language Model

## Tujuan

Memahami bagaimana teks diubah menjadi angka sehingga dapat diproses oleh model bahasa sederhana.
